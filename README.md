# 🤖 AI Prompts para Backend em Java

Repositório de prompts estruturados para uso com IA como copiloto no desenvolvimento backend com **Java 21 + Spring Boot**.

---

## 📖 Sobre o Projeto

Este projeto faz parte: 

| | |
|--|--|
| **Plataforma** | [DIO — Digital Innovation One](https://www.dio.me) |
| **Bootcamp** | Globant — Java & Spring Boot AI Developer |
| **Projeto:** | Usando IA Como Copiloto para Criar Novas Features no Seu Projeto |

O objetivo é demonstrar como utilizar IA de forma prática no dia a dia de desenvolvimento, apoiando:

- Implementação de código  
- Planejamento de features  
- Debug e resolução de problemas  
- Aprendizado contínuo  

---

## 🗂️ Estrutura do Repositório

```bash
prompts/
 ├── copiloto-java-agentcode.md
 ├── copiloto-java-ask.md
 ├── copiloto-java-modoPlan.md
 ├── copiloto-java-study.md
 ```

## 🧩 Modos de Uso

Os prompts foram organizados em quatro modos, cada um com um objetivo específico.

### ⚙️ Agent Code — Implementação

Arquivo: `copiloto-java-agentcode.md`  
[Ver prompt](prompts/copiloto-java-agentcode.md)

Responsável por gerar código completo e estruturado.

**Principais usos:**
- Criar APIs REST  
- Implementar CRUD  
- Gerar estrutura de projeto  

---

### ❓ ASK — Dúvidas e Debug

Arquivo: `copiloto-java-ask.md`  
[Ver prompt](prompts/copiloto-java-ask.md)

Focado em entendimento e diagnóstico.

**Principais usos:**
- Explicar erros (ex.: NullPointerException)  
- Entender código existente  
- Sugerir soluções sem implementar diretamente  

---

### 🧭 PLAN — Planejamento

Arquivo: `copiloto-java-modoPlan.md`  
[Ver prompt](prompts/copiloto-java-modoPlan.md)

Cria um plano antes da implementação.

**Principais usos:**
- Definir arquitetura  
- Planejar novas features  
- Identificar riscos e validações  

---

### 📚 STUDY — Aprendizado

Arquivo: `copiloto-java-study.md`  
[Ver prompt](prompts/copiloto-java-study.md)

Focado em aprendizado profundo.

**Principais usos:**
- Estudar conceitos de Java e Spring Boot  
- Entender boas práticas  
- Evoluir conhecimento técnico  

---

## 🚀 Como Usar

1. Escolha um modo (Agent, Ask, Plan ou Study)  
2. Acesse o arquivo correspondente  
3. Copie o prompt  
4. Cole por ex no: ChatGPT  
5. Descreva sua necessidade  

**Exemplos:**
- Criar uma API de usuários  
- Explicar um erro em Java  
- Planejar uma nova funcionalidade  
- Estudar conceitos do Spring Boot

## 🤖 Exemplos de Uso com IA

### ⚙️ Agent Code (Implementação)

- "Crie uma API REST de cadastro de usuários com Spring Boot"
- "Implemente um CRUD de produtos com validação de dados"
- "Adicione uma nova feature de pedidos em um sistema existente"

---

### ❓ ASK (Dúvidas e Debug)

- "Estou recebendo NullPointerException, o que pode ser?"
- "Qual a diferença entre @Service e @Component no Spring?"
- "Por que meu endpoint está retornando 404?"

---

### 🧭 PLAN (Planejamento)

- "Planeje uma API de autenticação com login e cadastro"
- "Como estruturar um projeto Spring Boot em camadas?"
- "Quero adicionar uma feature de carrinho de compras, como organizar?"

---

### 📚 STUDY (Aprendizado)

- "Me explique o que é Injeção de Dependência no Spring"
- "Como funciona o JPA no Java?"
- "Quando usar DTO em uma API?"

---

### 💡 Exemplos mais avançados

- "Quais são os trade-offs de usar JPA vs JDBC em um projeto Spring Boot?"
- "Como aplicar boas práticas de validação em APIs?"
- "Quais problemas podem acontecer se eu não tratar exceções?"
